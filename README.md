# butykos / Drinkustry

A 4-player asymmetric co-op cyberpunk-megacorp tycoon, built in Godot 4.2.

Pivoted in May 2026 from a single-player alcohol-tycoon prototype (`Alcohol Empire Tycoon`) into a four-corp co-op game (Agricultural / Industrial / Logistics / Business) where each corp has a signature mechanic of brewery-interior depth.

- **Engine:** Godot 4.2, GDScript
- **Main scene:** `scenes/ui/main_menu.tscn`
- **Repo branch model:** `main` (stable) ← `dev` (permanent) ← feature branches
- **Design direction:** see `design_docs/` (chronological)
- **Architecture + conventions:** see `CLAUDE.md`
- **Pre-pivot snapshot:** `DEVELOPMENT_STATUS.md` (archived)
